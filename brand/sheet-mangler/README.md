# Mangler brand

- Company: **Mangler**
- Product: **Sheet-Mangler** (alt: **Manglerrrr**)
- Focus: fixing spreadsheets and composing agentic workflows on top of them.
- Vibe: stark, surgical, hypermodern, lab-grade AI data wrangler.

## Files

- `theme.css` — color and typography tokens, base layout, UI kit, hero/sections.
- `landing.html` — minimal responsive landing page using the theme.
- `logo-wordmark-dark.svg` — wordmark for dark backgrounds.
- `logo-wordmark-light.svg` — wordmark for light backgrounds.
- `logo-mark-dark.svg` — square `[m]` mark for dark backgrounds.
- `logo-mark-light.svg` — square `[m]` mark for light backgrounds.
- `logo-wordmark-manglerrrr-alt.svg` — optional alt wordmark with extended “r”s.

## Color tokens

- Background: `#0B0D10`
- Surface: `#14171D`
- Border subtle: `#20242E`
- Border strong: `#2B323F`
- Text primary: `#E6EAEF`
- Text muted: `#9CA3AF`
- Accent: `#29E3FF`
- Accent weak: `rgba(41, 227, 255, 0.16)`

Optional alt accent palette is available by applying `.theme-magenta` on `body`.

## Typography

- Sans: Inter (or system UI), 400 / 600 weights.
- Mono: IBM Plex Mono (or system monospace).
- Sizes: 12, 14, 16, 20, 28, 40.

## Usage

- Open `brand/sheet-mangler/landing.html` in a browser to preview the landing page.
- Import `brand/sheet-mangler/theme.css` into your app or Storybook to reuse tokens and UI components.

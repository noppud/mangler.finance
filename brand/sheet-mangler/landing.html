How it works

Connect to live spreadsheets, repair them, and layer agentic workflows that watch for change and act on it in real time.

Ingest
Connect Google Sheets, Excel, or CSVs; sync live ranges without manual exports or copy-paste.
Profile
Detect schema, types, and anomalies with column-level stats across every tab.
Fix
Apply rules and agents to clean formats, coerce types, and repair broken formulas.
Verify
Run deterministic validators so every agent step is explainable, testable, and reversible.
Export / Orchestrate
Write back to sheets, trigger downstream agents, or emit clean tables into your existing stack.
Workflow DSL 
workflow "month-end-close" {
  sheet    source: "Finance/AR!A:F"
  watch    trigger: "on_edit"

  agent    name: "type-fixer",    task: "coerce amounts, normalize dates"
  agent    name: "anomaly-scout", task: "flag outliers > 3σ"

  validate rule: "total_debits == total_credits"
  writeback target: "Finance/AR_clean!A:F"
  notify   channel: "slack://#finance-ops"
}

FAQ

Will it modify my source?
By default Mangler writes to separate ranges or sheets; workflows can be configured to write back in-place when you’re ready.
How do I verify changes?
Every workflow logs before/after snapshots, constraint checks, and agent actions so you can diff what changed at the cell level.
What formats are supported?
Google Sheets, Excel, CSV, and TSV are first-class; warehouse tables and other sources are available via connectors.
